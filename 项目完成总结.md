### 1. 在 Railway 创建项目

1. 登录 Railway
2. 点击 "New Project"
3. 选择 "Deploy from GitHub repo"
4. 连接你的 GitHub 仓库

### 2. 添加 PostgreSQL 数据库

1. 在项目中点击 "New"
2. 选择 "Database" → "PostgreSQL"
3. 等待数据库创建完成
4. Railway 会自动设置 `DATABASE_URL` 环境变量

### 3. 执行数据库初始化

**方法 1：使用 Railway 控制台**
1. 点击 PostgreSQL 服务
2. 点击 "Data" 标签
3. 点击 "Query" 按钮
4. 粘贴 `database_schema.sql` 内容
5. 点击 "Run"

```bash

\i database_schema.sql

# 验证表创建
```

```sql
SELECT
  COUNT(CASE WHEN total_recharged > 0 THEN 1 END) * 100.0 / COUNT(*) as pay_rate
```

## 🎓 课程演示要点

### 1. 开发流程（7步法）

```
需求分析 → 技术选型 → 数据库设计 → 后端开发 → 前端开发 → 测试调试 → 部署上线
```

### 2. 为什么需要数据库？

**问题**："每个用户免费 3 次" —— 怎么记录？

- ❌ Session：关闭浏览器就重置
- ❌ 本地文件：多用户冲突
- ✅ 数据库：持久化 + 多用户安全

### 3. 充值模式的优势

- 用户体验好（充一次，多次用）
- 提高留存率（有余额就会继续用）
- 降低支付成本（减少支付次数）

### 4. 个人如何收款？

- 阶段 1：个人收款码 + 客服确认（现在）
- 阶段 2：接入第三方支付（码支付/虎皮椒）
- 阶段 3：企业资质 + 官方支付

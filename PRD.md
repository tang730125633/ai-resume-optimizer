# AI简历优化器 - 产品需求文档

## 1. 产品定位
- 目标用户：中年求职者（30-50岁）
- 核心价值：AI优化传统简历
- 收费模式：50-100元/次

## 2. 核心功能（MVP）
### 2.1 简历上传
- 支持格式：Word (.docx), PDF
- 文件大小：< 5MB

### 2.2 AI分析
- 识别简历内容
- 分析问题点：格式、内容、关键词
- 给出优化建议

### 2.3 生成优化版本
- 生成3个版本：
  - 版本1：传统格式优化
  - 版本2：现代简约风格
  - 版本3：ATS友好版本

### 2.4 下载
- 支持Word格式下载
- 包含优化说明

## 3. 技术栈
- 前端：HTML + Tailwind CSS
- 后端：Python Flask
- AI：Claude API
- 部署：Railway

## 4. 开发步骤
1. 创建项目结构
2. 开发前端上传页面
3. 开发后端API
4. 集成Claude AI
5. 实现文件处理
6. 部署上线

## 5. 成功标准
- 能上传简历
- AI能分析并给出建议
- 能生成优化版本
- 能下载结果
